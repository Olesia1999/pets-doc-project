# Проект

Добро пожаловать в документацию проекта!

Здесь вы можете найти информацию о том, как установить, запустить и настроить проект.

## Быстрый старт

1. Установите зависимости
2. Запустите локальный сервер документации: `mkdocs serve`
3. Откройте в браузере http://127.0.0.1:8000

## Структура документации

- `docs/` — страницы документации
- `mkdocs.yml` — конфигурация MkDocs
- `images` - картинки проекта
